## 📝 고객 요구사항 정리
1. 신용카드 등록 기능이 구현되어야 합니다. 이 기능은 중요
2. 카드 정보에 대한 보안은 철저해야 하며 8자리나 보안 코드는 작성될 때 숫자가 표시되지 않게 해야 합니다.
3. 여러 카드를 등록할 수 있어야 합니다.
4. 테스트 URL를 준비해야 합니다.

## 📋 기능 목록
- 카드 번호 input
  - `<CardNumberInput />`
- 여러 카드 정보 등록
- 배포 URL
